#  **Taskify** 

A modern task management (todo-style) application built with Next.js and React.  
This app allows users to create, organize, and manage tasks efficiently across boards and columns, providing a clean and intuitive workflow experience.

## **Key Features:**

-- Create, update, and delete tasks  

-- Organize tasks across boards and columns  

-- Drag and drop functionality for task management  

-- Clean and responsive user interface  

-- Client-side state management for smooth interactions  


## **Technologies Used:**

-- Frontend: Next.js (App Router), React, TypeScript, Tailwind CSS, shadcn/ui, @dnd-kit  

-- Backend & Services: Supabase (Database), Clerk (Authentication)
